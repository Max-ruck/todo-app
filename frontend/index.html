🔒 Datenschutzhinweis

Diese App speichert zur Nutzung folgende Daten auf dem Server des Betreibers:

  • Aufgaben, Archiv und Kommentare
  • Aktivitäts-Log (Erstellt, Erledigt, Gesendet …)
  • Session-Token im Browser (kein Tracking-Cookie)

Speicherort: Server des Betreibers  ·  Aufbewahrung: bis zur Account-Löschung
